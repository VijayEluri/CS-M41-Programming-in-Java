    1. This program takes a list of integers as parameters, and prints out "true" in case the list contains a duplicate, and "false" otherwise.
    2. Some examples: 
      > Duplicate 0 5 7
false
> Duplicate 0 0 1
true
    3. This program must not contain any comments.
  • Specification of MaxCount:
    1. This program takes a list of intergers as parameters, and prints the maximal number of times a number occcurs in that list.
    2. Some examples: 
      > MaxCount
0
> MaxCount 3 3 4
2
> MaxCount 1 2 3
1
    3. This program must not contain any comments.
  • Specification of Plateau:
    1. This program takes a list of integers as parameters, and prints the length of the longest contiguous sequence of equal values where the values of the elements just before and just after this sequence are (strictly) smaller (if such elements exist).
    2. Some examples: 
      > Plateau
0
> Plateau 1
1
> Plateau 1 1
2
> Plateau 1 1 2
1
> Plateau 1 1 2 2
2
> Plateau 1 1 2 2 3
1
> Plateau 1 1 2 2 0
2
> Plateau 0 1 2 3 2 2 2 2 3 3 2 1 0 -1 -1 0 0 0 -1 4
3
    3. This program may contain comments.
